# Tennis Game Score Board kata

A kata to practice using test doubles

inputs-> referee says which player has scored.
outputs-> current score 

Examples:
1 -> Fifteen Love
11 -> Thirty  Love
111 -> Forty  Love
1111 -> player 1 wons
2222 -> player 2 wons
2 -> Love Fifteen
1112 -> Forty Fifteen
121212 -> Deuce
1212121 -> Advantage player 1
