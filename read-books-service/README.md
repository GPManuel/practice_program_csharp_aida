# Read Books Service

A kata to practice TDD with test doubles.

INPUTS: USUARIO-DADO(DIRECTO), USUARIO-LOGEADO(INDIRECTO)
OUTPUTS: LISTA DE LIBROS (DIRECTO)
SIDE EFFECTS: 
	LOGED-USER, 
	OBTENCIÓN DE LOS LIBROS
	AMIGOS DEL LOGED-USER

Ejemplo2: LogedUser: Paco ---D Amigos:Juan ---D Usuario-dado: PEPE ---D Lista vacía
Ejemplo1: LogedUser: Paco ---D Amigos:Juan ---D Usuario-dado: JUAN ---D Libros de Juan 
Ejemplo3: LogedUser: NO ---D Usuario-dado: ANY ---D Emite ERROR
